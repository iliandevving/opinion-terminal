# Opinion Terminal

Opinion Terminal is the first professional interface for trading Opinion Prediction Markets like a professional trader.

## Features

- Real-time orderbook visualization
- Price charts with multiple timeframes
- Market browser organized by category (Sports, Crypto, Economics, etc.)
- Direct integration with Opinion API

## Tech Stack

- Next.js 15
- React 19
- Chakra UI v3
- TanStack Query
- lightweight-charts

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Environment Variables

Create a `.env.local` file with:

```
NEXT_PUBLIC_OPINION_API_KEY=your_api_key
```
